## 一、计划核心说明

+ **定位与目标** +
  
  适用于零基础运维小白，旨在通过 10 天学习掌握实习必备技能，成功获取运维 / 云原生 / DevOps 实习 offer。每天学习 5 小时，前 3 天补基础前提知识（网络 + Linux + 工具），后 7 天聚焦核心技术实操与项目落地，内容围绕 “能写进简历、能应对面试、适配实习工作” 设计。
+ **环境要求** +
  
  统一使用阿里云 ECS（2 核 4G，新用户免费），需提前开放特定端口，避免反复配置安全组。
+ **关键产出** +
  
  包括 6 个可写入简历的实战项目、3 份实用手册、2 个自动化 Shell 脚本以及 1 份适配实习的优化简历。

> [!NOTE]
> 家庭网络结构梳理与排查、Linux 系统初始化与用户环境搭建、远程服务器文件管理、Nginx 故障模拟与修复、Docker 与 K8s 部署 Nginx 对比、K8s 部署 PHP 动态应用

## 二、每日详细学习任务

<!-- tabs:start -->

#### **第 1 天**

计算机网络基础，目标是懂 TCP/IP 和网络排查。学习内容涵盖 TCP/IP 模型、核心概念（IP 地址、端口、DNS）、网络命令（ping、ip addr 等），当日项目为家庭网络结构梳理与排查，还包含 TCP 和 UDP 区别等面试考点。

#### **第 2 天**

Linux 系统原理与基础命令，目标是懂系统结构和核心操作。学习系统基础（文件系统结构、用户与权限）、文件管理命令（cd、ls 等）、用户与系统命令（useradd、passwd 等），当日项目为 Linux 系统初始化与用户环境搭建，涉及 Linux 核心目录作用等面试考点。

#### **第 3 天**

运维工具入门，目标是会用 SSH/Vim/ 文件传输。学习 SSH 远程连接（原理、工具、免密登录）、Vim 文本编辑（三种模式切换、常用命令）、文件传输工具（SCP 命令、可视化工具），当日项目为远程服务器文件管理，包含 SSH 连接服务器等面试考点。**Italian**

#### **第 4 天**

Web 服务部署与故障排查（Nginx 实战）。学习 Nginx 基础与部署（作用、安装、服务管理）、配置与页面自定义、故障排查体系，当日项目为 Nginx 故障模拟与修复，涉及 Nginx 安装启动等面试考点。

#### **第 5 天**

容器技术基础（Docker+K8s 集群搭建）。学习 Docker 基础（核心概念、与虚拟机区别、安装）、核心命令（镜像命令、容器命令）、K8s 集群搭建（作用、部署工具、单节点部署），当日项目为 Docker 与 K8s 部署 Nginx 对比，包含 Docker 镜像和容器关系等面试考点。

#### **第 6 天**

数据库容器化部署（MySQL 实战）。学习 K8s 部署 MySQL（特点、步骤）、进入容器与 SQL 操作（进入容器方法、基础 SQL 语法）、故障排查，当日项目为 MySQL 数据管理与故障排查，涉及 K8s 部署 MySQL 注意事项等面试考点。

#### **第 7 天**

监控系统搭建（Prometheus 实战）。学习 Prometheus 基础（作用、核心概念、部署工具）、部署（添加仓库、更新仓库、部署、查看状态）、暴露 UI 与指标查询，当日项目为 K8s 节点监控与报表生成，包含 Prometheus 监控常用指标等面试考点。

#### **第 8 天及以后**

从现有内容可知第 8 天开始综合项目实战，如 K8s 部署 PHP 动态应用等，后续将围绕更复杂的综合项目展开学习与实践。
